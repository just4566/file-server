# 文件访问服务

github地址：https://github.com/just4566/file-server

## 启动

```bash
# 安装依赖
npm install

# 启动项目
npm run serve
```



## 后续一键启动

项目目录下双击 `start.bat`，将在 CMD 窗口运行



## 共享文件位置

项目目录下 => static 文件夹



## 如何访问

- http://localhost:5000/[文件名.扩展名]

- https://localhost:5001/[文件名.扩展名]

## 图片收集到本地

- http://localhost:5000 网络图片收集页
